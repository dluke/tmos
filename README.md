

# Twitching Motility on Surfaces (TMoS)  

Simulation tool for studying twitching motility of bacteria.   

Written by Daniel Barton   

## Requirements  

### C++ libraries  
Python 3.8 (python3-dev package for Python.h)  
Boost 1.67   
Eigen 3.3+  
Nlopt (linux: libnlopt-dev package)  
VTK 7+ (optional)  

### Python analysis  
Numpy 1.9+  
Scipy  
Matplotlib  

pandas  
seaborn  
tqdm  
tabulate  
SALib
